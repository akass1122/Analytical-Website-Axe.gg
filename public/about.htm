About

~~~
rift.gg is a personal side project of mine.

The site provides statistics for all champions of League of Legends, including win rates, ban rates, pick rates, most effective skills, masteries, etc. with live-updated data.

For the client I use JavaScript and jQuery to make a dynamic website.

For the server I use Node.js, Express.js, and REST API.

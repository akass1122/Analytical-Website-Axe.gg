About

~~~
Vocane.com is a personal side project of mine.

I made this web application to search for Volta charging stations. It uses raw data from Volta public API and Google API to show closest electric car charging stations to the address, entered in the search box, or all charging stations of a chosen city. It displays these stations on Google Maps and also shows them in a table.
